独立项目-Toyester - Independent Project - Toyester

一款类幸存者游戏，旨在通过装备BD，赋予刷子游戏更多的策略集选择和目标感，目前正在筹备taptap测试。

A game similar to Survivor, aiming to provide the brush game with more strategic options and a sense of purpose through BD equipment. Currently preparing for taptap testing.

GJ项目-无妄箓 - GJ Project - Wuwang Lu

一款横版解密闯关游戏，GJ作品，主题为二元性。通过阴阳二元的不同世界联动，回归至一元的本体通关。

A side-scrolling decryption and clearance game, GJ work, with a theme of duality. Through the linkage of yin and yang dualism in different worlds, return to the essence of unity to pass.

嵌入式项目-基于可见光通信的智能导览系统 - Embedded Project - Intelligent Guidance System Based on Visible Light Communication

通过可见光频闪和摄像机帘式快门特性，构建出的可见光通信交互方案。实现移步换景，阅后即焚，文件加密传输等多重功能。
基于可见光适宜以一对多的单向传输或以一对一的双向传输特性，设置了博物馆这一应用场景。该项目共获两项专利

An interactive solution for visible light communication, created through visible light frequency flashing and camera curtain-shutter characteristics. It achieves features like changing scenes with each step, self-destructing messages, and encrypted file transmission. Suitable for one-to-many unidirectional or one-to-one bidirectional transmission using visible light, with a focus on museum applications. The project has obtained two patents.

独立项目-代行 - Independent Project - Daixing

一款融合了DBG机制和身份推理类游戏的电子桌游，该作品最早是想尝试当时刚兴起的AI绘画。并制定了中式克苏鲁的美术基调，最终在完成部分程序工作后由于UI美术原因弃坑。

An electronic board game that combines DBG mechanics and identity deduction gameplay. The initial idea for this project was to experiment with AI art, which was just emerging at the time. The art style was influenced by Chinese Lovecraft, but the project was eventually abandoned due to UI art reasons after completing some programming work.

桌游项目-火种 - Board Game Project - Spark

一款聚焦于土地革命时期，反围剿战争的DBG+战棋机制融合桌游，该项目目前由上海市党校进行研讨推广，因此受到过青年报采访。

A board game that focuses on the Land Revolution period and integrates DBG and war chess mechanics. The project is currently being promoted and discussed by the Shanghai Party School and has been interviewed by the Youth Daily.
